# phoennixai-platform

Static, single-file HTML dashboards for the **PhoennixAI Platform** — a React + GSAP UI prototype that visualizes the PhoennixAI ecosystem (agents, routines, skills, and connectors).

## What’s inside
- **index.html** — main platform dashboard (self-contained).
- **PhoennixPro_v5_FINAL.html** — packaged “final” build of the platform dashboard.
- **DocumentHub_PhoennixAI_v6.html** — Document Hub dashboard/page.
- **ULTRAPLAN_PhoennixAI_Business_Operations.html** — ULTRAPLAN / business operations dashboard/page.
- **phoennixai_project_instructions.md** — project instructions, standards, and ecosystem reference.

## How to view
### Option A: GitHub Pages (recommended)
Enable **Settings → Pages** and set the source to the `main` branch.

### Option B: Open locally
Download/clone the repo and open `index.html` in a browser.
> Note: Some browser security settings can affect local file access; hosting (Pages) is the smoothest experience.

## Tech
- HTML (self-contained)
- React 18 (CDN / UMD)
- GSAP animations

## Status
Prototype / concept UI (static front-end; no backend included in this repo).
